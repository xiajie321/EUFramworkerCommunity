# EU 备忘录 (EU Memo)

一个基于 UIToolkit 的完善备忘录工具，提供便捷的记录、管理和查看功能，帮助开发者高效管理项目任务和灵感。

## 功能特点

- **快速创建**：一键创建新备忘录，自动生成时间戳。
- **实时保存**：所有修改实时保存到本地，无需手动操作。
- **搜索过滤**：支持按标题和内容快速搜索备忘录。
- **界面美观**：基于 UIToolkit 开发，提供现代化的暗色主题界面。
- **数据持久化**：数据保存在 Application.persistentDataPath 中，不会污染项目资源。

## 使用方法

1. 打开菜单 `EUFarmworker -> Extension -> EU 备忘录`。
2. 点击左上角的 `+` 按钮创建新备忘录。
3. 在右侧编辑区输入标题和内容。
4. 使用搜索框快速查找历史记录。
5. 选中备忘录后点击底部的 `删除` 按钮可移除不需要的记录。

## 文件结构

- `ConfigPanel/`: 包含 UI 布局 (UXML) 和样式 (USS) 文件。
- `Script/`: 包含核心逻辑代码。
- `Doc/`: 文档说明。
- `extension.json`: 扩展描述文件。

## 依赖

- Unity 2021.3 或更高版本
- UIToolkit
