# EUFramework Core MVC 更新日志

本文档记录了 EUFramework Core MVC 模块的所有重要更改。

---

## [1.0.6] - 2026-03-02

### 优化

- **生命周期调整**：优化了架构的初始化与销毁顺序，解决模块间依赖可能导致的空引用问题。
  - 初始化顺序调整为：`Utility` -> `Model` -> `System` (基础设施 -> 数据 -> 逻辑)
  - 销毁顺序调整为：`System` -> `Model` -> `Utility` (逻辑 -> 数据 -> 基础设施)
- **内存优化**：优化 `TypeEventSystem` 的清理逻辑，在 `Clear` 时重置过大的事件数组，防止极端情况下的内存滞留。

### 文档

- **性能原理解析**：补充关于 `TypeEventSystem` 在 Mono (JIT) 和 IL2CPP 下的性能优化原理说明，详细解释了静态泛型缓存机制。

---

## [1.0.5] - 2026-03-02

### 新增

- **程序集定义**：新增 Assembly Definition 文件 (`.asmdef`)，方便进行代码裁切和编译优化。

---

## [1.0.4] - 2026-02-21
